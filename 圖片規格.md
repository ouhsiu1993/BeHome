| 圖片用途 | 建議尺寸 (px) | 比例 | 檔案大小 | 格式 |

| Hero 背景圖 | 1920 x 1080 | 16:9 | < 500KB | WebP / JPG |
| 產品圖片 (3張) | 800 x 600 | 4:3 | < 200KB | WebP / JPG |
| 核心技術圖片 | 800 x 1000 | 4:5 | < 200KB | WebP / JPG |
| 關於我們圖片 | 800 x 1000 | 4:5 | < 200KB | WebP / JPG |

## 更新圖片後

將圖片放入 `public/` 資料夾後，需更新 `src/App.tsx` 中的圖片路徑：

```tsx
// Hero 背景
src="https://images.unsplash.com/..." → src="/hero-bg.webp"

// 產品圖片 (PRODUCTS 陣列)
image: 'https://images.unsplash.com/...' → image: '/product-air-one.webp'

// 核心技術
src="https://images.unsplash.com/..." → src="/tech-section.webp"

// 關於我們
src="https://images.unsplash.com/..." → src="/about-team.webp"
```
